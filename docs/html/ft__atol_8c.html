long ft_atol ( char *  str)

Converts a string to an long integer.

Parses the string str, skipping any leading whitespace, then reads an optional sign (+ or -), followed by numerical digits, converting them into an integer value.

Parameters
strPointer to the null-terminated string to convert.
Returns
The converted integer value. If the string does not contain any valid integer, returns 0.
