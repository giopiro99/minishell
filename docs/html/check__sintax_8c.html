bool check_sintax ( t_token *  tokens)

Checks the syntax of the entire token list.

Skips space tokens at the start of the list. If the list starts with a pipe, prints a syntax error. Then iterates over all tokens and checks syntax via ft_sintax_errors.

Parameters
tokensPointer to the first token in the list.
Returns
true if a syntax error was detected, false if syntax is correct.
