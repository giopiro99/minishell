Correctly assigns types to tokens related to redirections and here-documents.

If the redirection flag is active, searches for the next ARGUMENT token and marks it as FILE_. If the token is HEREDOC, searches for the next ARGUMENT token and marks it as DELIMITER. Also manages activation/deactivation of redir_flag and is_first flags.

Parameters

tokensCurrent token to process.
