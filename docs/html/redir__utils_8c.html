t_token *  tokens,
t_cmd *  cmd,
t_type  type 
)

Gets the filename from a token based on redirection type and assigns it to the command.

Finds the token representing a FILE_ or DELIMITER (for heredoc) and assigns the filename to the command:

  • REDIR_OUT or REDIR_APPEND uses ft_get_outfile_name()
  • REDIR_IN assigns cmd->input to the filename
  • HEREDOC assigns cmd->input to "temp_file.txt" (temporary file used for heredoc)
Parameters
headhead of the token list
tokenspointer to the current token (not used as input but reset to head)
cmdcommand structure to assign the filename to
typeredirection type (REDIR_IN, REDIR_OUT, REDIR_APPEND, HEREDOC)
Returns
always returns 0
