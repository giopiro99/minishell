int ft_atoi ( const char *  nptr)

Converts a string to an integer.

Parses the string nptr, skipping any leading whitespace, then reads an optional sign (+ or -), followed by numerical digits, converting them into an integer value.

Parameters
nptrPointer to the null-terminated string to convert.
Returns
The converted integer value. If the string does not contain any valid integer, returns 0.
