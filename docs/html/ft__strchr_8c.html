Locates the first occurrence of a character in a string.

Searches the string 's' for the first occurrence of the character 'c'. If 'c' is found, returns a pointer to its position in the string. If 'c' is the null terminator '\0', returns a pointer to the end of the string. Returns NULL if the character is not found.

Parameters
sThe null-terminated string to search in.
cThe character to locate (interpreted as unsigned char).
Returns
char* Pointer to the first occurrence of 'c' in 's', or NULL if not found.
