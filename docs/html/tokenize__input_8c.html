void ft_define_tokens ( t_token *  tokens)

Assigns types to tokens and applies command-specific rules.

This function processes the token list to:

  • assign types (e.g., ARGUMENT, REDIRECTION, etc.),
  • mark special spaces,
  • correctly merge tokens in export and echo commands.

t_token * tokenize ( char *  input)

Tokenizes the entire user input string.

Allocates a temporary buffer, repeatedly calls ft_get_tokens to build the token list, then calls ft_define_tokens to finalize processing.

Parameters
