Copies up to size - 1 characters from the string src to dst, null-terminating the result.

Copies characters from 'src' to 'dst' until either 'size - 1' characters are copied or the null terminator of 'src' is reached. The destination string is always null-terminated if 'size' is greater than 0. Returns the total length of 'src'; this allows detection of truncation by comparing the return value with 'size'.

Parameters
dstDestination buffer where the string is copied.
srcSource null-terminated string to copy from.
sizeTotal size of the destination buffer.
Returns
