Concatenates 'src' to the end of 'dst' with size limit.

Appends string 'src' to the end of 'dst', ensuring the total length does not exceed 'size' (including null terminator). Returns the total length of the string it tried to create, i.e., initial length of 'dst' plus length of 'src'. If 'size' is less than or equal to initial length of 'dst', returns 'size' plus length of 'src' without concatenation.
