void ft_merge_tokens_echo ( t_token **  tokens)

Handles merging multiple -n, -nn, etc. tokens for the echo command.

If the first token is "echo" and the next one is a -n option, merges all subsequent -n* options into a single token. This allows correct handling of syntax like echo -n -nn hello.
