void ft_assign_spaces ( t_token *  prev,
t_token *  tokens,
int *  between_arg 
)

Assigns the token type for space tokens based on surrounding context.

  • If the previous token is an ARGUMENT and is a '-n' sequence, sets the current token as GARBAGE.
  • If the previous token is ARGUMENT, ENV_VAR, or QUOTE, and the next token is also one of those, and no space has been assigned between arguments yet, sets the current token as SPACES, otherwise sets it as GARBAGE.
  • In all other cases, sets the token type as GARBAGE.
Parameters
prevPrevious token (can be NULL)
tokensCurrent token to modify
between_argPointer to an integer flag indicating if a space between arguments has already been found (0 or 1)
