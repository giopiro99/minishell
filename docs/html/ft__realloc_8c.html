Reallocates a NULL-terminated matrix of strings to a new size.

Allocates a new matrix of size 'new_size' and copies up to 'old_size' existing strings from the original matrix using deep copies (ft_strdup). Frees the old matrix after copying. The new matrix is NULL-terminated.

Parameters
matrixThe original NULL-terminated matrix of strings.
new_sizeThe new desired size (number of rows) for the matrix.
old_sizeThe number of strings to copy from the original matrix.
Returns
char** A pointer to the newly allocated and copied matrix, or NULL on failure.
